# 用語集

ドキュメント・会話で使う呼称を統一する。ゲーム内UIの表記とズレやすい点を明示する。

## 育成ウマ娘まわり

| 用語 | 意味 | データソース |
|------|------|----------------|
| **育成ウマ娘所持スキル** | 育成ウマ娘の**覚醒レベル（カード開花）**を上げることで取得できるスキル。育成中のSP購入リストに載る対象 | mdb `available_skill_set` → `characters.json` の `skillsByAwakening` |
| **育成ウマ娘覚醒レベル** | カード開花の段階（mdb 上は `need_rank`）。本ツールは**最大想定**で全ランクの所持スキルを合算 | `skillsByAwakening` のキー `"1"`〜 |

> **注意（旧称）**: 以前のドキュメントでは「覚醒スキル」などと書いていたが、ゲーム内の「覚醒進化」と混同しやすいため、上記の **育成ウマ娘所持スキル** に統一する。

## スキル進化・固有

| 用語 | 意味 | 本ツールでの扱い |
|------|------|------------------|
| **覚醒進化** | ゲーム内用語。**金→金**の進化（例: シナリオ進化で `全身全霊` → `ベストを尽くして！`） | 購入リスト（`available_skill_set`）には**進化前の金**のみ入る。実機確認済み。明示の置換ロジックは不要 |
| 白→金 | 白スキル下位を持つ金スキル（`lowerSkillId`） | 一覧では白行を隠し、金行に白+金の合算コスト（`goldLower.js`） |
| **○→◎** | 同一グループの下位○と上位◎（`groupRate` 順）。ヒントは○に付き◎専用ヒントは無い | ○ヒントのみでも **◎行に繰り上げ**、cost(○)+cost(◎)。金が計画にあれば ○+◎+金 |
| **×（劣位）** | `group_rate < 0` のスキル（例: 冬ウマ娘×） | 購入チェーンに含めない。ガチ想定ではヒント対象外 |
| **固有スキル** | 育成ウマ娘本体のユニークスキル（例: `I Never Goof Up!`） | `skill_set` にあり **購入リスト外**。extract 経路では SP 対象に入らない |
| **継承固有** | 継承で入る固有の汎用枠（UIオプション） | 個数×baseSp の手動 ON 行。固有スキル本体とは別 |

| フィールド | ドキュメント上の呼称 |
|------------|---------------------|
| `characters.json` → `skillsByAwakening` | 育成ウマ娘所持スキル（覚醒レベル別） |
